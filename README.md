# 🛒 Интернет-магазин «TechStore»

> Учебный проект по дисциплине «Командная строка и веб-технологии»  
> Практическая работа №3

## 📋 Описание проекта

**cli-pract3** — это учебный веб-проект, представляющий собой прототип интернет-магазина электроники, одежды и книг. Проект разработан в образовательных целях для закрепления навыков вёрстки, стилизации и программирования на стороне клиента.

### 🔹 Основные возможности

- 📦 **Каталог товаров** с фильтрацией по категориям:
  - Электроника
  - Одежда
  - Книги
- 🛒 **Корзина покупок** с функциями:
  - Добавление товаров
  - Просмотр выбранных позиций
  - Удаление отдельных товаров
  - Очистка корзины
  - Оформление заказа (имитация)
- 🎨 **Адаптивный дизайн** с использованием чистого CSS
- ⚡ **Интерактивность** на чистом JavaScript (без внешних зависимостей)

## 🗂 Структура проекта

```
cli-pract3/
├── index.html          # Главная страница
├── catalog.html        # Страница каталога с фильтрами
├── products.html       # Страница отдельного товара
├── cart.html           # Страница корзины покупок
├── style.css           # Таблицы стилей
├── script.js           # Логика приложения (корзина, фильтры)
└── img/                # Изображения товаров и элементы интерфейса
```

## 🛠 Технологии

| Технология | Назначение |
|------------|------------|
| **HTML5** | Разметка структуры страниц |
| **CSS3** | Стилизация и адаптивный дизайн |
| **JavaScript (ES6+)** | Интерактивность, управление корзиной, фильтрация |

## 🚀 Запуск проекта

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/OtradinskiyIvan/cli-pract3.git
   cd cli-pract3
   ```

2. Откройте файл `index.html` в браузере:
   ```bash
   # Через файловый менеджер
   # Или с помощью локального сервера, например:
   python3 -m http.server 8000
   ```

3. Перейдите в браузере по адресу: `http://localhost:8000`

## 🎓 Образовательные цели

Проект предназначен для отработки следующих навыков:

- ✅ Работа с DOM API
- ✅ Обработка событий пользователя
- ✅ Управление состоянием приложения (корзина)
- ✅ Организация многофайловой структуры веб-проекта
