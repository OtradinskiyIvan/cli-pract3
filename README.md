# Интернет-магазин товаров

Этот проект представляет собой простой интернет-магазин с каталогом товаров, фильтрацией по категориям, подробными описаниями товаров и функционалом корзины.

## Функционал

### Основные возможности:
- **Главная страница** (`index.html`): Введение в магазин.
- **Каталог товаров** (`catalog.html`): Просмотр всех товаров с возможностью фильтрации по категориям (Электроника, Одежда, Книги).
- **Страница товара** (`products.html`): Подробное описание выбранного товара с характеристиками, изображением и кнопкой добавления в корзину.
- **Корзина** (`cart.html`): Просмотр добавленных товаров, удаление, очистка корзины и оплата.

### Функции корзины:
- Добавление товаров в корзину.
- Удаление товаров из корзины.
- Очистка всей корзины.
- Оплата (симуляция).
- Сохранение состояния корзины в localStorage.

### Фильтрация:
- Фильтр по категориям на странице каталога.
- Отображение количества товаров в выбранной категории.

## Как запустить проект

1. Скачайте или клонируйте репозиторий.
2. Откройте файл `index.html` в любом современном веб-браузере.
3. Переходите по ссылкам в меню для навигации по сайту.

Проект использует только HTML, CSS и JavaScript, поэтому не требует сервера или дополнительных зависимостей.

## Технологии

- **HTML5**: Структура страниц.
- **CSS3**: Стилизация и адаптивность.
- **JavaScript (ES6+)**: Логика корзины, фильтрации и взаимодействия с пользователем.
- **localStorage**: Сохранение состояния корзины.
