# Интернет-магазин «TechStore»

> Учебный проект по дисциплине «Командная строка и веб-технологии»  
> Практическая работа №3

## Описание проекта

**cli-pract3** — учебный веб-проект, представляющий прототип интернет-магазина электроники, одежды и книг. Проект разработан в образовательных целях для закрепления навыков вёрстки, стилизации и клиентского программирования.

### Основные возможности

- Каталог товаров с фильтрацией по категориям:
  - Электроника
  - Одежда
  - Книги
- Корзина покупок с базовым функционалом:
  - Добавление и удаление товаров
  - Просмотр выбранных позиций
  - Очистка корзины
  - Имитация оформления заказа
- Адаптивный дизайн на чистом CSS
- Интерактивность на нативном JavaScript без внешних зависимостей

## Структура проекта

```
cli-pract3/
├── index.html          # Главная страница
├── catalog.html        # Страница каталога с фильтрами
├── products.html       # Страница отдельного товара
├── cart.html           # Страница корзины
├── style.css           # Таблицы стилей
├── script.js           # Логика приложения (корзина, фильтрация)
└── img/                # Статические ресурсы (изображения)
```

## Используемые технологии

| Технология | Назначение |
|------------|------------|
| HTML5 | Семантическая разметка страниц |
| CSS3 | Стилизация и адаптивная вёрстка |
| JavaScript (ES6+) | Управление состоянием, обработка событий, фильтрация данных |

## Запуск проекта

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/OtradinskiyIvan/cli-pract3.git
   cd cli-pract3
   ```

2. Откройте проект в браузере одним из способов:
   - Откройте файл `index.html` напрямую через файловый менеджер
   - Запустите локальный сервер:
     ```bash
     python3 -m http.server 8000
     ```
     Затем перейдите по адресу: `http://localhost:8000`

## Образовательные цели

Проект предназначен для отработки следующих компетенций:

- Работа с DOM API и манипуляция элементами страницы
- Обработка пользовательских событий
- Управление состоянием приложения (корзина покупок)
- Организация модульной структуры веб-проекта
- Стилизация интерфейса с использованием современных возможностей CSS
- Реализация клиентской фильтрации и поиска данных

## Требования к окружению

- Современный веб-браузер (Chrome 90+, Firefox 88+, Edge 90+, Safari 14+)
- Текстовый редактор или IDE (по желанию)
- Локальный веб-сервер (рекомендуется для корректной работы относительных путей)
